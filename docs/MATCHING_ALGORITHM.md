# マッチングアルゴリズムの説明

このドキュメントでは、親子マッチングアプリで使用しているマッチングスコアの計算方法について説明します。

## スコア計算の概要

入力した情報（生年月日・氏名（ひらがな）・出身地）をもとに、親子双方の情報を比較しマッチングスコアを自動計算します。

### 📊 マッチングスコアの計算方法

- 🎂 **生年月日**：
  - 年月日一致で80点
  - 月日一致70点
  - 年月一致60点
  - 年齢差が近い場合は最大+5点加算
- 👤 **氏名（ひらがな）**：
  - 名字・名前とも一致10点
  - 名前のみ一致7点
  - 名字のみ一致3点
- 📍 **出身地**：
  - 都道府県・市区町村とも一致10点
  - 都道府県のみ一致7点
- ⚖️ **スコア合算**：
  - 親→子（60%、合計60点満点）
  - 子→親（40%、合計40点満点）
  - 合計100点満点

## ポイント

すべての情報が必須ではありません。覚えている範囲で入力することで、少しずつマッチング精度が向上します。わからない項目は空欄のままでも問題ありません。
